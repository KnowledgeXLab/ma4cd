# Genomics Research Skill (instance)

MA4CD skill **instance** for genomics / omics data source discovery.  
Uses the same rule-pack schema as other skills — see [skills/README.md](../README.md).

## Activate

```bash
python main_workflow.py --skill genomics-research "寻找基因组与组学开放数据"
```

Domain-specific content lives only under `rules/` (NCBI, Ensembl, GEO, SRA, etc.).
